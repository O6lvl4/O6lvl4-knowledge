[[vrm|VRM]] モデルの共有プラットフォーム。pixiv が運営。[[vroid-studio|VRoid Studio]] とは別サービス。

## VRoid Studio との違い

- **VRoid Studio** = 作るツール
- **VRoid Hub** = 共有するプラットフォーム

## 機能

- 自作モデルの公開・他人モデルのダウンロード
- **利用条件メタデータ** — 作家が「商用利用 OK / 改変 OK / 暴力表現 NG」等を細かく設定。VRM のライセンスメタデータ機能がそのまま活きる
- pixiv FANBOX / BOOTH 連携で有料配布も可能
- 対応アプリ（VRChat, cluster, 3tene 等）から直接読み込み可能

## 他人のモデルを使うとき

VRoid Hub からダウンロードしたモデルは **そのモデルに設定された利用条件に従う**。自分で作ったモデルとはライセンス状況が全く異なるので注意。詳細は [[vroid-studio-license|VRoid Studio ライセンス]] を参照。
