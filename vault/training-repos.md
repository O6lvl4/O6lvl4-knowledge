プログラミング学習・演習用リポジトリ群。テスト駆動形式の問題集や、フレームワーク習得用テンプレートなど。

## JavaScript / TypeScript

### js-training
JavaScript 学習用問題集の個人 fork。`aid-on-js-training` ベース。
https://github.com/O6lvl4/js-training

### aid-on-js-training
JavaScript 初学者向けの段階的学習教材。テスト駆動 (TDD) 形式で 000-039 の段階別問題を自動採点。基礎・中級・応用に分かれた構成。
https://github.com/Aid-On/aid-on-js-training

### next-pet-diary
`create-next-app` で生成した Next.js のペット日記練習プロジェクト。
https://github.com/Aid-On/next-pet-diary

## iOS / Swift

### AIChatPractice
SwiftUI で AI チャット UI を実装する練習用 Xcode プロジェクト。
https://github.com/Aid-On/AIChatPractice

### sandbox-ios-app
SwiftUI ベースのモダン iOS アプリテンプレート。Swift PM、Harmonize による architectural lint、fastlane match + TestFlight の自動 CI/CD を備える。
https://github.com/Aid-On/sandbox-ios-app

## Minecraft / Bots

### mine-rabbit-bots
Mineflayer 専用のサーバー検証 sandbox。Paper サーバー（オフライン認証）上でプラグインや bot 設定を安全に実験するための環境。Craftsman 本体から独立。
https://github.com/Aid-On/mine-rabbit-bots

## CLI / LLM

### llmine
ターミナルから OpenAI / Azure OpenAI / Anthropic / AWS Bedrock / Ollama を統一インターフェースで叩く CLI。pipe 入力対応、多言語 UI。
https://github.com/O6lvl4/llmine
