Aid-On 向けの請求書ジェネレーター。単一 HTML ファイルで完結。

## 構成

`index.html` 一枚に HTML / CSS / JS を内包したシングルファイル構成。左に入力フォーム、右にプレビューを並べる 2 カラムレイアウトで、ブラウザだけで請求書を生成できる。
