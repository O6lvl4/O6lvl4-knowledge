日本語でプログラムを記述する[[stack-oriented-programming|スタック指向]][[programming-language|プログラミング言語]]。

## プログラマ向けの一言

**[[forth|Forth]] を日本語化した言語。** 「3と 4を 足す」のように日本語の語順（SOV = 主語-目的語-動詞）が逆ポーランド記法とそのまま一致するところを活かしている。

## なぜ日本語とスタック指向が合うか

```
英語:   add 3 and 4        → 動詞が先（前置記法）
数学:   3 + 4              → 演算子が中（中置記法）
日本語: 3と 4を 足す        → 動詞が後（後置記法 = 逆ポーランド記法）
Forth:  3 4 +              → 後置記法
```

日本語の自然な語順がそのまま RPN になるため、スタック指向と日本語は構造的に相性がいい。

## コード例

```
3と 4を 足す。        \ → 7
5を 二乗する。        \ → 25
「こんにちは」を 表示する。
```

## 他の日本語プログラミング言語

- **なでしこ** — Web/教育向け。日本語で手続き型プログラミング
- **ひまわり** — なでしこの前身
- **プロデル** — Visual Basic 風の日本語プログラミング

Mind はこれらと違い、[[forth|Forth]] 直系のスタック指向という点が特徴的。
