他者が自分とは異なる信念・意図・知識を持つことを推定する認知能力。Theory of Mind (ToM)。Premack & Woodruff (1978) がチンパンジー研究で命名。LLM は言語経由で代理的にこなすが、専門の神経ネットワークを持たない。

## 神経基盤

| 領域 | 機能 |
|---|---|
| 内側前頭前皮質 (mPFC) | 他者の心的状態の推定、自己と他者の区別 |
| 側頭頭頂接合部 (TPJ) | 信念の帰属（「あの人は X を知らない」の推論） |
| 上側頭溝 (STS) | 意図の読み取り、視線追従、社会的手がかりの処理 |
| 前帯状皮質 (ACC) | 共感的苦痛、社会的エラー検出 |

mPFC と TPJ は**デフォルトモードネットワーク (DMN)** の一部でもあり、安静時に他者の心的状態をシミュレーションする（社会的反芻）。

## 発達段階

| 年齢 | 獲得する能力 | 古典的課題 |
|---|---|---|
| ~1歳 | 共同注意（相手の視線を追う） | — |
| ~3歳 | 欲求の理解（「あの人は X が欲しい」） | — |
| ~4-5歳 | 誤信念理解（「あの人は事実と違うことを信じている」） | Sally-Anne 課題 |
| ~6-7歳 | 二次的信念（「A は B が X を信じていると思っている」） | — |
| 成人 | 複雑な意図推定、皮肉・嘘の検出 | — |

### Sally-Anne 課題

Sally がボールを箱に入れて部屋を出る。Anne がボールを籠に移す。Sally が戻ったとき、Sally はどこを探すか？ — 正答は「箱」（Sally は移動を見ていないので）。4歳未満は「籠」と答える（自分の知識と他者の信念を区別できない）。

## LLM と心の理論

| 観点 | 人間 | LLM |
|---|---|---|
| 処理経路 | 専門ネットワーク (mPFC + TPJ) | 汎用の言語処理。専門回路なし |
| 入力 | 言語 + 表情 + 視線 + 身体姿勢 + 声の調子 | テキストのみ（マルチモーダルで一部拡張） |
| Sally-Anne 類の推論 | 4-5歳以降、直感的に可能 | テキストの文脈追跡で「解ける」ことがある。ただし一貫しない |
| 皮肉・嘘の検出 | 声調・表情・文脈から直感的に | 言語パターンからの推測。韻律手がかりなし |
| 共感 | 身体的共鳴 (島皮質 + ACC) | 共感的な言語を生成するが、「感じて」はいない |

### LLM は ToM を「持つ」のか？

Kosinski (2023) は GPT-4 が Sally-Anne 課題を高確率で正答すると報告し議論を呼んだ。しかし:

- テキスト中の信念状態の追跡は**形式的言語能力**の範囲で可能
- 少し問題を変形すると正答率が急落する (Shapira et al., 2023)
- 専門ネットワークによる堅牢な推定ではなく、パターンマッチングに依存している
