[[copyright-law|著作権]]を保持したまま、一定の条件で他人に利用を許諾する標準ライセンス体系。略称 CC。

## 仕組み

著作権法のデフォルトは「全部ダメ（All Rights Reserved）」。CC ライセンスはそこから「これは OK」と条件を明示して開放する。

## ライセンスの構成要素

4つの条件の組み合わせで6種類 + CC0:

| 記号 | 名前 | 意味 |
|---|---|---|
| BY | 表示 | クレジットを表示せよ |
| SA | 継承 | 同じライセンスで配布せよ（GPL に似ている） |
| NC | 非営利 | 商用利用禁止 |
| ND | 改変禁止 | 改変禁止 |

## 6種類のライセンス（制限の緩い順）

| ライセンス | 商用 | 改変 | 条件 |
|---|---|---|---|
| CC BY | OK | OK | クレジット表示 |
| CC BY-SA | OK | OK | クレジット表示 + 同じライセンスで公開 |
| CC BY-NC | NG | OK | クレジット表示 |
| CC BY-NC-SA | NG | OK | クレジット表示 + 同じライセンスで公開 |
| CC BY-ND | OK | NG | クレジット表示 |
| CC BY-NC-ND | NG | NG | クレジット表示（最も制限が強い） |

## CC0（シーシーゼロ）

上記6つとは別枠。著作権を **可能な限り放棄** する宣言。

- 権利者がいなくなる → 取り消し不可能
- 誰でも何にでも使える。クレジット表示すら不要
- パブリックドメインに近い状態を作る

### CC0 と使用許諾の違い

| | CC0 | 通常の使用許諾（VRoid プリセット等） |
|---|---|---|
| 権利者 | いない（放棄済み） | 元の作者が保持 |
| 取り消し | 不可能 | 理論上は可能（規約変更） |

実例: Unsplash はもともと CC0 だったが、再配布業者対策で独自ライセンスに変更した。CC0 時代の写真は取り消せないが、新規写真は新ライセンスが適用される。

## よくある誤解

- **「フリー素材」= CC0 ではない。** 多くのフリー素材は CC BY（クレジット必要）や独自ライセンス
- **CC ライセンスは取り消せない。** 一度 CC で公開した著作物はその条件で永久に利用可能（ただし CC0 以外は条件違反で権利主張できる）
- **CC は著作権を放棄しない。**（CC0 を除く）著作権を持ったまま利用条件を緩和するもの
