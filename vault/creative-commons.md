「この作品、こういう条件なら使っていいよ」を世界共通のマークで示す仕組み。略称 CC。

## なぜ存在するか

[[copyright-law|著作権法]]のデフォルトは **「全部ダメ」**。イラストを描いた瞬間に「無断転載禁止、改変禁止、商用禁止」が自動でかかる。

でも「クレジットさえ入れてくれれば自由に使っていいよ」と思う作者もいる。そのとき、いちいち利用規約を自分で書くのは大変だし、読む側も毎回違うルールを確認するのは面倒。

CC は **条件を4つのマークの組み合わせで表現** することで、この問題を解決した。

## 4つのマーク

| マーク | 意味 | わかりやすく |
|---|---|---|
| **BY** | 表示 | 「私の名前を書いてね」 |
| **SA** | 継承 | 「あなたも同じ条件で公開してね」 |
| **NC** | 非営利 | 「お金儲けには使わないでね」 |
| **ND** | 改変禁止 | 「そのまま使ってね、変えないでね」 |

## 6種類（制限の緩い順）

| ライセンス | 商用 | 改変 | 必要なこと |
|---|---|---|---|
| CC BY | OK | OK | クレジット表示だけ |
| CC BY-SA | OK | OK | クレジット表示 + 同じ条件で公開 |
| CC BY-NC | NG | OK | クレジット表示 |
| CC BY-NC-SA | NG | OK | クレジット表示 + 同じ条件で公開 |
| CC BY-ND | OK | NG | クレジット表示 |
| CC BY-NC-ND | NG | NG | クレジット表示（一番厳しい） |

## CC0（シーシーゼロ）— 特別枠

上の6つとは全く違う。**「この作品の権利を放棄します。誰のものでもありません。」** という宣言。

- クレジット表示すら不要
- 何に使ってもいい
- **一度放棄したら取り消せない**（ここが最大のポイント）

### CC0 と「フリー素材」の違い

「フリー素材サイト」のほとんどは CC0 ではない。権利者が「使っていいよ」と許諾しているだけで、権利を放棄してはいない。

| | CC0 | よくあるフリー素材サイト |
|---|---|---|
| 権利者 | いない（放棄済み） | サイト運営者 or 作者 |
| 取り消し | **絶対にできない** | 規約変更で制限が増える可能性あり |
| 例 | — | いらすとや、VRoid プリセット |

**実例: Unsplash。** 写真共有サイト Unsplash はもともと CC0 で写真を提供していた。しかし写真を丸ごとダウンロードして別サイトで売る業者が現れたため、CC0 をやめて独自ライセンスに変更した。CC0 時代の写真は取り消せないが、以降の写真は新ルールになった。

## よくある誤解

- **「無料 = 何してもいい」ではない。** 無料でも「クレジット必須」「商用禁止」の条件がつくことは多い
- **CC ライセンスは撤回できない。** 一度 CC BY で公開した作品はその条件で永久に使える
- **CC0 以外は権利を放棄していない。** 条件に違反したら著作権者は権利を主張できる
