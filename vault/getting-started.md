## Writing Notes

Notes are written in Markdown using Obsidian. You can use:

- **Wikilinks**: `[[Page Name]]` to link between notes
- **Tags**: in frontmatter YAML
- **Frontmatter**: YAML metadata at the top of each file

## Publishing

Notes with `public: true` in frontmatter (default) are published to the viewer.
Set `public: false` to keep a note private.
