AITuber の配信を回すために必要なランニングコストの構造分析。コストは大きく **LLM（対話生成）** と **TTS（音声合成）** の 2 つで構成される。

## コスト構造

```
ユーザー入力（コメント等）
    ↓
  LLM（対話生成）  ← コスト①
    ↓ テキスト
  TTS（音声合成）  ← コスト②
    ↓ 音声
  配信出力（OBS 等）← 固定費（サーバー/回線のみ）
```

LLM と TTS の 2 つが従量課金で、配信基盤自体のコストはほぼ無視できる。

## LLM コスト

1 分あたりのやりとりで $0.02（Haiku 4.5 の場合）を基準とした試算。

| モデル | 入出力単価比 (対 Haiku) | 1 時間あたり | 月間 (毎日 1h) |
|---|---|---|---|
| **Haiku 4.5** | 1x | $1.20（≒ ¥180） | ≒ ¥5,400 |
| **Sonnet 4.6** | ~4x | ~$4.80（≒ ¥720） | ≒ ¥21,600 |
| **Opus 4.6** | ~20x | ~$24.00（≒ ¥3,600） | ≒ ¥108,000 |

AITuber 用途では Haiku が圧倒的にコスパが良い。応答品質は配信のカジュアルなやりとりには十分。

## TTS コスト

日本語の発話速度（約 300〜400 文字/分）を前提に、AI が半分の時間しゃべると仮定すると **1 時間あたり約 10,000 文字**（30 分発話 × 350 文字/分）。

| TTS サービス | 単価 | 1 時間あたり | 月間 (毎日 1h) |
|---|---|---|---|
| **[[aivishub\|AivisHub]]** | 0.044 cr/文字（1 cr = ¥1） | ¥440 | ¥13,200 |
| **Google Cloud TTS** (WaveNet) | ¥2,400/100 万文字 | ≒ ¥24 | ≒ ¥720 |
| **Amazon Polly** (Neural) | ¥2,400/100 万文字 | ≒ ¥24 | ≒ ¥720 |
| **Azure TTS** | ¥2,400/100 万文字 + 無料枠 | ≒ ¥24 | ≒ ¥720 |
| **[[voicevox\|VOICEVOX]]** (ローカル) | ¥0（電気代のみ） | ¥0 | ¥0 |
| **[[style-bert-vits2\|Style-Bert-VITS2]]** (ローカル) | ¥0（電気代のみ） | ¥0 | ¥0 |

AivisHub はクラウド TTS の約 18 倍の単価。ただしアニメ声系の音声品質は AivisHub / VOICEVOX が上。Google / AWS / Azure は英語品質は高いが日本語キャラクター音声には向かない。

## 合算コスト（1 時間配信）

Haiku + 各 TTS の組み合わせ：

| 構成 | LLM | TTS | 合計/時間 | 月間 (毎日 1h) |
|---|---|---|---|---|
| Haiku + AivisHub | ¥180 | ¥440 | **¥620** | ¥18,600 |
| Haiku + クラウド TTS | ¥180 | ¥24 | **¥204** | ¥6,120 |
| Haiku + VOICEVOX (ローカル) | ¥180 | ¥0 | **¥180** | ¥5,400 |

## コスト支配要因

AivisHub を使う場合、コスト比は **TTS 7 : LLM 3** で TTS が支配的。VOICEVOX に切り替えるだけでコストが 1/3 以下になる。

コスト削減の効果が大きい順：

1. **TTS をローカル実行に切り替える**（VOICEVOX / Style-Bert-VITS2） → TTS コスト ¥0 化、最大のインパクト
2. **発話量を減らす**（短い応答、間を作る、テンポ設計） → TTS が従量の場合に直接効く
3. **LLM モデルの選択** → Haiku なら既に最安ライン。Sonnet 以上は 4 倍〜20 倍

TTS をローカル化した場合、LLM コストだけが残り **毎日 1 時間配信で月額 ¥5,400**。これは個人 AITuber にとって十分現実的なライン。

## 前提と留意事項

- 為替レートは $1 = ¥150 で計算
- LLM コスト（$0.02/分）は 2026 年 5 月時点の Haiku 4.5 実測値
- API 料金は変動するため、具体的な数字は時点依存
- 画像生成・リアクション等の追加機能を入れるとコストが上乗せされる
- VPS / GPU サーバーを使う場合はインフラ固定費が別途発生

## 押さえどころ（カード化候補）

- AITuber 配信コストの 2 大構成要素 → **LLM（対話生成）と TTS（音声合成）の従量課金。配信基盤自体のコストはほぼ無視できる**
- Haiku 4.5 の AITuber コスト → **$0.02/分 = $1.20/時間 ≒ ¥180/時間。毎日 1 時間配信で月額 ¥5,400**
- AivisHub の TTS 単価 → **0.044 cr/文字（1 cr = ¥1）。1 時間配信で約 ¥440。クラウド TTS（Google/AWS/Azure）の約 18 倍**
- AITuber コスト最適化の最大レバー → **TTS をローカル実行（VOICEVOX / Style-Bert-VITS2）に切り替えること。TTS コスト ¥0 化で合計コストが 1/3 以下に**
- AivisHub 利用時のコスト比 → **TTS 7 : LLM 3 で TTS が支配的。発話量削減が直接コスト削減に効く**
- 最安構成の月額 → **Haiku + VOICEVOX（ローカル）= 毎日 1 時間配信で月額 ¥5,400**
