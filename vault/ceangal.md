GPU ネイティブの UI フレームワーク。[[snaidhm]] 上のレイアウト・ウィジェット・インタラクション層。[[almide|Almide]] で実装。

## 機能

- **Flexbox レイアウト** — Yoga 互換（flex, gap, wrap, 絶対配置, パーセンテージサイズ）
- **GPU レンダリング** — Canvas 2D ゼロ、計算シェーダベース
- **仮想リスト** — O(1) スクロール
- **リアクティブ状態管理**
- **DOM オーバーレイ**
