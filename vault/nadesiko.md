日本語で記述する手続き型[[programming-language|プログラミング言語]]。クジラ飛行机が開発。

## プログラマ向けの一言

**日本語の文章がそのままプログラムになる言語。** [[mind|Mind]] がスタック指向なのに対し、なでしこは手続き型で、プログラミング初学者・教育用途に向いている。

## コード例

```
「こんにちは」と表示。
3に4を足して表示。

もし、3＞2ならば
  「大きい」と表示。
違えば
  「小さい」と表示。
ここまで。

10回
  「繰り返し」と表示。
ここまで。
```

## バージョン

- **なでしこ v1** — Windows デスクトップアプリ（Delphi 製）
- **なでしこ v3** — JavaScript / Node.js 製。ブラウザで動く。教育現場での採用が進む
